Steps running the JAR file.
1- Open your CMD
2- go to the project directory
3- run "Java -jar goeuro $COUNTRY_NAME"
4- a CSV file will be generated with the country name.

Steps running the source code
1- This is a maven project, so make sure you have maven installed.
2- Import the project into eclipse.
3- Force update on maven to download the GSON library.
4- configure run configurations to add the country name as a parameter when running
5- Run Source.Java as starting point for your application
